Sobre o Projeto

O projeto BlogDog foi desenvolvido durante a disciplina de Trabalho Interdisciplinar II: Banco de Dados e Sistemas Inteligentes na PUC Minas. Como o nome da disciplina sugere, foram usados conceitos das matérias de Banco de Dados e Introdução a Sistemas Inteligentes para a realização do trabalho, além de Algoritmo e Estrutura de Dados II com ênfase em Java. O projeto é uma continuação de um trabalho desenvolvido na primeira disciplina de Trabalho Interdisciplinar I e dessa vez o foco é o desenvolvimento de uma aplicação completa utilizando backend.
